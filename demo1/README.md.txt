# Web básica para restaurante – Demo

Este proyecto es una demo de una web mínima para restaurantes
que no tienen web o solo usan redes sociales.

## Objetivo
Crear una web clara y funcional que permita:
-posicionamiento seo
-usabilidad 
-performance
- Ver el menú
- Llamar fácilmente
- Escribir por WhatsApp
- Encontrar el local

## Enfoque
- Mobile-first
- Diseño simple
- Carga rápida
- Sin elementos innecesarios

## Tecnologías
- HTML
- CSS
-php
-javascript
